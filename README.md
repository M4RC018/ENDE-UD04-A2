# ENDE-UD04-A2

Repositorio correspondiente a la actividad 2 de la unidad didáctica 4 de Entornos de Desarrollo.

## Descripción

Este proyecto contiene el código fuente del proyecto Figuras desarrollado en Java.  
El objetivo es practicar el uso de Git y GitHub realizando las siguientes tareas:

- Clonación de repositorios
- Control de versiones
- Uso de ramas
- Refactorización de código
- Trabajo colaborativo

## Tecnologías utilizadas

- Java
- Git
- GitHub
